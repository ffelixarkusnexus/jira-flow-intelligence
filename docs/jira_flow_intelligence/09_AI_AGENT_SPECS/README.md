# AI Agent Specifications

## Purpose

Define how AI is used within the system.

---

## Role of AI

- Translate structured insights → natural language
- Improve clarity and usability

---

## NOT the Role

AI does NOT:
- compute metrics
- detect bottlenecks
- make decisions

## Key Principle

AI enhances understanding  
It does not replace logic
