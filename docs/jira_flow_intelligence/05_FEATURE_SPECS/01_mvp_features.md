# MVP FEATURES — EXACT

## 1. Time-in-Status

For each issue:
- breakdown of time spent in each status

---

## 2. Cycle Time

- total duration from created → done (or now)

---

## 3. Bottleneck Detection

- multi-signal scoring (NOT just max time)
- outputs:
  - stage
  - reasons
  - confidence

---

## 4. Daily Insight

System generates:
- 1–3 top insights

Example:
"Review is slowing down delivery due to increased wait time and growing queue."

---

## 5. Alerts

Rules:
- status duration exceeded
- cycle time exceeded

---

## 6. Minimal UI

Single dashboard:
- top insight
- bottleneck
- alerts
